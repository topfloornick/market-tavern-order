# Market Tavern - Online Ordering Demo

Modern online ordering website for **Market Tavern** — an upscale New American restaurant in Stockton, CA.

## Features

- **Full Indexable Menu** — Every dish is on the page (not hidden in a PDF), so Google can find "best brisket sandwich near me" and show Market Tavern
- **Online Ordering** — Customers order directly, no 25-30% DoorDash fees
- **Mobile-First Design** — Smooth experience on any device
- **Cart System** — Add items, adjust quantities, checkout for pickup
- **SEO Optimized** — Proper meta tags, semantic HTML, fast load times
- **Modern Aesthetic** — Dark upscale theme matching Market Tavern's brand

## Tech Stack

- React 18 + Vite
- React Router for navigation
- Context API for cart state management
- CSS-in-JS with inline styles
- Google Fonts (Playfair Display + Inter)

## Run Locally

```bash
npm install
npm run dev
```
